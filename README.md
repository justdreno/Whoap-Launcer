## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/whoap.git
cd whoap

# Install dependencies
npm install
```

### Development

Run the dual-process dev server (React + Electron):

```bash
npm run dev
```

### Build

Create a production-ready installer:

```bash
npm run build
```
