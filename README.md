## Quick Start

### 1. Installation

```bash
# Clone the repo
git clone https://github.com/yourusername/whoap.git
cd whoap

# Install dependencies
npm install
```

### 2. Development

Launch the development environment (React + Electron):

```bash
npm run dev
```

### 3. Build

Create a production-ready installer for your OS:

```bash
npm run build
```
